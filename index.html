Comandos de programação

Nesse site, irei explicar os comandos de programação.

HTML: (h1)

É uma tag html usada para definir o cabeçalho de nível 1 em um documento. Geralmente é usado para títulos principais.

(p)

Essa tag é usada para definir um parágrafo de texto.

ing

Seletores e propriedades de ajustes.

CSS: formatação de cor

Usamos a propridade "color" para o texto e "background-color" para o fundo.

centralização

Para centralizar elemento, podemos usar a propriedade "text-aligh".

tamanho

Para definir o tamanho do texto, usamos a propriedade "font-size", e para um bloco usamos "width" e "height".
